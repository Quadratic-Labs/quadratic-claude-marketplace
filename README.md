# Quadratic Claude Marketplace

This is a curated collection of Claude Code plugins for Quadratic projects.

## Available Plugins

- DevKit - Development Workflow Plugin: Comprehensive development workflow automation for software releases, testing, and quality checks.

## Requirements

- Claude Code CLI
- Git
- GitHub CLI (`gh`) - for milestone checking and release management
- Optional: `yq` for better YAML parsing (fallback uses grep)
